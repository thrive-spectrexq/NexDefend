# NexDefend Frontend

The NexDefend Frontend is a React-based single-page application that provides the user interface for the NexDefend platform.

## Features

-   **Dashboard**: Provides a centralized view of the security posture of the organization.
-   **Incident Management**: Allows users to view, create, and manage security incidents.
-   **Vulnerability Management**: Allows users to view, create, and manage vulnerabilities.
-   **Asset Management**: Allows users to view, create, and manage assets.
-   **Reporting**: Provides various reporting capabilities.
