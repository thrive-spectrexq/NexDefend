# NexDefend Cloud Connector

The NexDefend Cloud Connector is a dedicated service for ingesting security logs and events from cloud providers (e.g., AWS CloudTrail, Azure Monitor, Google Cloud Logging) and SaaS applications (e.g., Okta, Google Workspace, Microsoft 365).

## Features

-   **Cloud Provider Integration**: Ingests security logs and events from various cloud providers.
-   **SaaS Application Integration**: Ingests security logs and events from various SaaS applications.
-   **Centralized Data Ingestion**: Provides a centralized point for ingesting cloud and SaaS data into the NexDefend platform.
